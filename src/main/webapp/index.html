HttpServletResponse 주요 메서드

  • setStatus(int sc) HTTP 상태 코드를 200 OK로 설정
  • sendError(int sc, String msg) 404 에러 페이지 전송
  • setHeader(String name, String value) 단일 헤더 설정 (덮어쓰기)
  • addHeader(String name, String value) 동일한 이름의 헤더 여러 개 추가
  • setContentType(String type) Content-Type을 JSON 등으로 지정
  • getWriter() 텍스트 응답 작성 (PrintWriter)
  • sendRedirect(String location) 클라이언트에게 특정 경로로 리다이렉트 지시
  • addCookie(Cookie cookie) 쿠키 설정 후 전송
